# Quick Start Guide

After extracting the archive:

1. Make scripts executable:
   chmod +x install.sh start.sh stop.sh

2. Run installation:
   ./install.sh

3. Start the application:
   ./start.sh

4. Open browser:
   http://localhost:8000

5. To stop:
   ./stop.sh

# Or one-liner after extract:
chmod +x *.sh && ./install.sh && ./start.sh
