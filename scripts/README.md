# Utility Scripts

Collection of utility scripts for managing AI Code Executor.

## Scripts

### install.sh
Automated installation script for setting up the application.

```bash
./scripts/install.sh
```

### start_https.sh
Start the application with HTTPS support.

```bash
./scripts/start_https.sh
```

### stop.sh
Stop all running containers.

```bash
./scripts/stop.sh
```

### generate_ssl.sh
Generate self-signed SSL certificates for HTTPS.

```bash
./scripts/generate_ssl.sh
```

## Main Start Script

The main start script is in the root directory for easy access:
```bash
./start.sh
```

This keeps the most commonly used script readily available.
